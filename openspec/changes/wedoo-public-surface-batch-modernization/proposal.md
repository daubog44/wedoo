## Why

Wedoo's public marketing surfaces are not yet visually coherent as a family. The homepage has been modernized again, but `/info`, `/candidato`, and `/azienda` still need to be assessed and potentially rebuilt as a single editorial batch so the public side stops feeling like mixed-era prototypes.

## What Changes

- map the public marketing routes as one coherent family
- modernize `/info`, `/candidato`, and `/azienda` using Figma plus `DESIGN.md`
- fix layout shifts, cramped canvases, and stale shells where they still appear
- keep CTA logic, route structure, and backend-like data intact
- stabilize route-level parity on section or canvas targets where full-page VRT is too fragile

## Non-Goals

- redesign auth or portal routes in this change
- introduce a new design system package as a global dependency
- replace Figma with third-party aesthetic libraries
