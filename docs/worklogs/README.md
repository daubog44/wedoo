# Worklogs

Questa cartella contiene i worklog di sessione del Ralph Loop.

Convenzione:

- una sottocartella per giorno: `YYYY-MM-DD`
- uno o piu file per sessione: `HHMM-task-slug.md`

Scopo:

- tracciare cosa ha fatto il loop nel tempo
- registrare problemi, fix, test e decisioni
- mantenere `prd.md` pulito come backlog
